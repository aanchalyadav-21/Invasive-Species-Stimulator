Ecosphera

Simulating Nature. Shaping the Future.

Ecosphera is an interactive simulator that demonstrates how invasive species spread across a habitat grid. It helps visualize how intervention changes the outcome and how ecological control can reduce infestation. Built as a polished academic project, it is designed for both learning and presentation.

Project Overview

📊
