Color Guide

Green

Healthy habitat. Default state before infection spreads.

Red

Infected cell. Red spreads to nearby green cells during the simulation.

Blue

Controlled cell. Blue can be placed on red or green cells and never turns green again.

How to use

1. Select a speed tab first. 2. Press Start Simulation. 3. Click any cell to paint it blue. 4. The spread stops automatically when no more spread is possible.
